# Configuração do Supabase — Studio JM

Este documento explica como configurar o Supabase para rodar o Studio JM localmente ou em produção.

## Serviços utilizados

- Supabase Database
- Supabase Auth
- Supabase Storage
- Row Level Security

## Estrutura necessária

### Tabelas

- `briefings`
- `projetos`
- `novidades`

### Buckets

- `portfolio`
- `briefings_attachments`

## Credenciais

No Supabase, acesse:

```txt
Project Settings > API
```

Copie:

```txt
Project URL
anon public key
```
